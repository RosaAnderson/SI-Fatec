Página com links

 Bem vindo à minha página de demonstração dos exercícios realizados
no curso Sistemas para Internet segundo semestre 2022.

 Esta página abriga principalmente e como objetivo principal os links
para as demais páginas construídas a partir e no decorrer das aulas.

 É uma página simples porém útil para navegar diretamente para as
páginas que criamos para exercitar os conhecimentos adquiridos em
durante a apresentação dos conceitos e comandos mas também fora dela.

 Usei o a tag de comando padrão de chamada de links aliada a tag para
criar um botão para dar um visual diferenciado.

 Nessas páginas ainda estamos trabalhando apenas a estrutura do HTML
portanto as configurações visuais presentes nessas páginas foram feitas
diretamente no HTML isso por que ainda não estamos usando CSS.

- Para fazer chamada de links basta usar a instrução a seguir:

 <a href="endereço_destino" > </a>

- Ou como eu usei no menu a instrução a seguir:

 <a href="endereço_destino" > <button" > Texto do Botão </button> </a>
